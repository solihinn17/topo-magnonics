## Directory Structure

This repository contains computational simulation data and scripts classified into the following folders based on the calculated properties:

* **`charge-berry/`**
    * Contains `.dat` files storing **charge Berry curvature** data.
* **`electronic-properties/`**
    * Contains `.gnu` files for **band structure** visualization, as well as data for **DOS** (Density of States) and **pDOS** (Projected Density of States).
* **`magnetic-exchange/`**
    * Contains `.out` files generated by **TB2J**. These files include magnetic exchange interaction parameters, specifically:
        * $J_{\text{iso}}$ (Isotropic exchange)
        * $J_{\text{aniso}}$ (Anisotropic exchange)
        * DMI (Dzyaloshinskii-Moriya Interaction)
* **`optical-conductivity/`**
    * Contains `.dat` files for **optical conductivity** data.
* **`spin-berry/`**
    * Contains `.dat` files storing **spin Berry curvature** data.
